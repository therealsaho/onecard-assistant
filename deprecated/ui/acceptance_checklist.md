# UI Acceptance Checklist

- [ ] **Chat Persistence**: Chat history remains visible after new messages are added.
- [ ] **Debug Panel**: Toggling "Show Debug Info" reveals router classification and tool details.
- [ ] **Confirmation Visuals**: Confirmation prompts are visually distinct (e.g., bold, colored border).
- [ ] **Voice Placeholder**: A microphone icon or "Voice Input" placeholder is visible.
- [ ] **Error Handling**: Simulated errors (e.g., network fail) show a non-breaking red banner.
- [ ] **Loading State**: "Processing..." spinner appears during orchestrator calls.
